# Statistical Reporting Checklist

- State the primary hypothesis and outcome.
- State sample sizes per group.
- Name the test or model.
- State whether assumptions were checked.
- Report effect size.
- Report confidence interval or credible interval.
- Report exact p-values where applicable.
- Describe missing data handling.
- Describe outlier handling.
- State multiplicity correction if multiple comparisons were tested.
- Make clear which analyses were exploratory.
- Attach or cite the script and input data path when possible.
